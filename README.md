# 띵보 (dding-bo)

띵타이쿤 게임을 위한 보조 프로그램입니다.

## 📖 소개

띵보는 띵타이쿤 게임을 플레이하는 사용자들을 위한 데스크톱 보조 프로그램입니다. 요리 계산, 강화 계산, 게임 정보 관리 등 다양한 기능을 제공하여 게임 플레이를 더욱 편리하게 만들어줍니다.

## ✨ 주요 기능

### 🏠 메인 페이지
- **요리 가격 초기화 타이머**: 1일과 3의 배수 날짜(1, 3, 6, 9, 12, 15, 18, 21, 24, 27, 30일) 새벽 3시 초기화까지 남은 시간 표시
- **야생 초기화 타이머**: 매주 수요일 오후 1시 초기화까지 남은 시간 표시
- **음신 순위**: 현재 가격이 최고가 대비 높은 순으로 정렬된 요리 순위 (상위 3개)

### 🍳 요리 계산기
- 요리 제작에 필요한 재료와 비용 계산
- 스킬 레벨에 따른 수확량 및 드롭률 계산
- 요리별 수익성 분석
- 씨앗, 베이스, 재료 관리

### ⚡ 강화 계산기
- 아이템 강화에 필요한 재료 및 비용 계산

### 📋 요리 정보
- 모든 요리 레시피 정보 확인
- 요리 등급 및 카테고리별 분류
- 재료별 요리 검색

### 🗺️ 지도
- 게임 지도 정보 제공

### ⚙️ 설정
- 앱 설정 관리
- 튜토리얼 관리
- 데이터 업데이트 설정

## 🛠️ 기술 스택

- **프레임워크**: Electron, Vue.js 3
- **빌드 도구**: Vite
- **스타일링**: Tailwind CSS, PrimeVue
- **상태 관리**: Pinia
- **라우팅**: Vue Router

## 📦 설치 방법

### 개발 환경 설정

1. 저장소 클론
```bash
git clone [repository-url]
cd dding
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm start
```

### 빌드

1. 패키지 빌드
```bash
npm run package
```

2. 설치 파일 생성
```bash
npm run make
```

## 🚀 사용 방법

1. 앱을 실행하면 메인 페이지가 표시됩니다.
2. 사이드바에서 원하는 기능을 선택합니다.
3. 각 기능 페이지에서 필요한 정보를 입력하고 결과를 확인합니다.
